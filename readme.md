## Installation

### [Oh My Zsh](http://ohmyz.sh)

```sh
wget -O $ZSH_CUSTOM/themes/common.zsh-theme https://raw.githubusercontent.com/OMRIFIJI/common-zsh-theme/refs/heads/master/common.zsh-theme
```

Update your `.zshrc` file with:
```sh
ZSH_THEME="common"
```
